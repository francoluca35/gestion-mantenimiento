También podrá restaurar una copia de seguridad desde un archivo de copia de seguridad realizado de forma externa por su administrador de base de datos. Para ello, presione el botón Examinar y luego seleccione el archivo de copia de seguridad (tenga en cuenta que la ruta seleccionada debe ser relativa al servidor). Una vez seleccionado, presione el botón Agregar y el archivo se añadirá a la lista de Copias de Seguridad del sistema.
Una vez seleccionada la copia de seguridad a restaurar, haga clic sobre el botón Confirmar.
Para poder restaurar una copia de seguridad correctamente es necesario el acceso exclusivo a la base de datos, por lo que todas las instancias del sistema deben estar cerradas al momento de la operación de restauración.
