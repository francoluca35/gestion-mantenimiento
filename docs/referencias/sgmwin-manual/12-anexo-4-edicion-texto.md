## Edición rápida de texto
SGMWin – Sistema de Gestión del Mantenimiento
EDICIÓN RÁPIDA DE TEXTO
Se puede dar el caso tenga que repetir un texto varias veces lo cual puede ser engorroso, para evitar ello puede seleccionar el texto a copiar de dos formas

## Las 2 formas de realizar esta operación son las siguientes:
La primera es, pintando todo el texto, arrastrando el mouse sobre el mismo con el botón izquierdo presionado, una vez realizado esto y sin despintar la selección, presione las teclas Ctrl+C para copiarlo.
La segunda, haga un clic con el botón derecho del mouse sobre el texto a copiar, de esta forma aparecerá un menú contextual donde seleccionará la opción Seleccionar todo, verá que queda marcado todo el texto, presione las teclas Ctrl+C para copiarlo o volviendo a hacer clic con el botón derecho del mouse, elija la opción Copiar del menú contextual
Después de realizar cualquiera de las dos opciones el texto seleccionado se copiará en el portapapeles de Windows, donde quedará disponible. Para poder obtenerlo debe presionar las teclas Ctrl+V o haga un clic con el botón derecho del mouse y seleccione la opción Pegar.
